# 前言

欢迎来到基于SSM的选题管理系统项目！本项目旨在帮助高校或研究机构方便、高效地管理课题选题过程。以下是对本项目的一些基本介绍。

# 内容介绍

基于SSM（Spring、SpringMVC、MyBatis）的选题管理系统，提供了一套完整的课题选题、审核、分配及管理流程。系统主要包括以下功能模块：

1. 用户模块：负责用户的注册、登录、信息修改等操作。
2. 选题模块：允许学生浏览课题列表，提交选题申请，教师可以查看选题情况，进行课题分配。
3. 审核模块：教师可以对学生提交的选题进行审核，通过或不通过。
4. 管理模块：管理员可以管理课题、用户和选题数据，进行系统设置等。

# 技术介绍

- 语言：Java
- 使用框架：Spring、SpringMVC、MyBatis
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16
